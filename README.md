# 2022-12-31

## 运行环境

- `python3`
- `bs4`
- `lxml`
- `requests`

## 运行方式

### 如有运行环境

```shell
python picture.py
```

### 如没有运行环境（须有anaconda）

在`cmd`中

1. ```shell
   conda create -n lsp beautifulsoup4 lxml requests python=3.9 -y
   ```

2. ```shell
   conda activate lsp
   ```

3. ```shell
   python picture.py
   ```

### 注意

为了环境配置顺利，建议不要科学上网
